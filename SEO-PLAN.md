# SEO Plan — Public Results Pages

## Phase 1 (Tech — ผมทำได้)
- /results — list ผลล่าสุดทุกหวย + CTA LINE
- /results/[lottery] — ประวัติงวด เช่น /results/thai
- /results/[lottery]/[date] — ผลงวดเดียว
- /stats/[lottery] — top numbers, frequency
- /howto, /lottery-schedule
- sitemap.xml (auto-gen ทุกผลใหม่)
- robots.txt
- Structured Data Schema.org (LotteryResult)
- Meta + Open Graph (LINE/FB share)
- Internal linking + breadcrumbs
- CTA "เปิด LINE @042xplcj" ทุกหน้า

## Phase 2 (Content — auto-generate)
- หน้า "แนวทาง" จากสถิติ (top เลข + เลขห้าม)
- บทความรายเดือน
- FAQ

## Phase 3 (บอสทำ)
- ซื้อ domain (300 บาท/ปี)
- Google Search Console signup (ฟรี)
- Google Analytics 4 (ฟรี)
- Submit sitemap

## Timeline
- เดือน 1-2: Tech + Google index เริ่ม
- เดือน 3-6: ติด long-tail keyword (50-5k traffic/mo)
- เดือน 6-12: 5k-100k traffic/mo (ถ้า content + backlinks)

## Estimated convert
- ~0.5% ของ traffic → กด LINE OA → add → ซื้อ
- 50k traffic → 250 ลูกค้าใหม่/เดือน

## Niche ที่สู้ได้
- Long-tail (เช่น "ตรวจหวยลาว 12 มิ.ย. 2569")
- หวยหุ้นต่างประเทศ (sanook/kapook ไม่เน้น)
- "เรทหวย" / "แนวทาง" (intent ตั้งใจซื้อ)
