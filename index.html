📜 กติกา Affiliate

ค่าคอม · วิธีแชร์ลิงก์ · ข้อตกลง

💸 ค่าคอมที่คุณจะได้

  • 20-40% ของยอดแทงสุทธิจากลูกค้าที่คุณชวนเข้ามา (ดูเปอร์เซ็นต์ของคุณในหน้ารวม)
  • คำนวณจาก ยอดรับ − รางวัลที่จ่ายแล้ว ต่องวด — ลูกค้าถูกเยอะ ค่าคอมก็ลดตาม
  • จ่ายให้ผ่านการโอนเงินตามรอบที่ตกลง (ดูประวัติได้ในหน้ารวม)

🔗 วิธีหาลูกค้าใหม่

  1. คัดลอกลิงก์ชวนของคุณจากหน้ารวม (ปุ่ม 📋 คัดลอกลิงก์)
  2. ส่งให้คนที่อยากแทงหวยกับเรา ผ่าน LINE OA ของคุณเอง
  3. เมื่อเขาเปิดลิงก์ + add เพื่อนกับ LINE OA หลักของเรา + คีย์เลข — ระบบจะ ผูกเขาเป็นลูกค้าของคุณอัตโนมัติ
  4. ลูกค้าใหม่ของคุณจะปรากฏในหน้า 👥 ลูกค้า

✍️ การคีย์บิลแทนลูกค้า

  • คุณคีย์บิลแทนลูกค้าได้ในหน้า ✍️ คีย์บิล (เช่นเขาส่งเลขมาที่ LINE ของคุณ)
  • บิลที่คุณคีย์จะถูกผูกกับ หัวบ้านของคุณอัตโนมัติ — admin จะเห็นว่ามาจากใคร
  • เลือกลูกค้าจาก dropdown ก่อนคีย์ (มีแต่ลูกค้าของคุณเท่านั้น)
  • ถ้าลูกค้าไม่เคยมีในระบบ → ให้เขาเปิดลิงก์ของคุณก่อน เพื่อสร้างบัญชีให้เขาก่อน

⚠️ ข้อตกลง

  • คุณ คีย์บิลแทนลูกค้าของตัวเองเท่านั้น ห้ามคีย์ให้ลูกค้าคนอื่น
  • ห้ามแก้ไข/ลบบิลของลูกค้าคนอื่น
  • การชวนต้องเป็นไปอย่างสุจริต — ห้ามใช้บัญชีปลอม / กดซ้ำเพื่อปั่นยอด
  • ถ้าตรวจพบทุจริต ระบบจะหักค่าคอมและระงับบัญชี Affiliate
